# astrbot-code-inviter

AstrBot 多码池加好友发码插件项目。

本仓库用于承载 `astrbot_plugin_code_inviter` 的后续开发。目标是在 AstrBot + NapCat / OneBot 11 场景下，通过群触发、一次性好友验证 token、私聊领取和管理员导入导出能力，实现多码池邀请码 / 兑换码发放。

## 当前状态

当前仓库处于项目初始化阶段，已经保存产品需求和开发拆分，但尚未实现插件业务逻辑。

已确认的 v1 范围见 [docs/prd.md](docs/prd.md)。

## 计划范围

- 多码池配置、触发词和领取规则。
- 群触发后生成一次性好友验证 token。
- 好友申请自动审批校验。
- 私聊领取单个未使用码。
- 文本列表和 CSV 批量导入。
- 库存、统计、查询和 CSV 导出管理命令。
- SQLite 本地持久化。

## 暂不包含

- 独立 Web 管理后台。
- 多机器人共享库存。
- 支付、订单、抽奖、积分等活动系统。
- 管理员补发功能。
